1. PROJECT DESCRIPTION:
This project uses a Computational Linguistics approach to evaluate 
Machine Translation (MT) errors in English-Vietnamese translations, 
specifically focusing on Passive Voice and Relative Clauses.

2. SYSTEM REQUIREMENTS:
- Python 3.9+
- Libraries: spacy, pandas, matplotlib, spacy-transformers

3. INSTALLATION (IMPORTANT):
Please run these commands in your terminal:
   pip install spacy pandas matplotlib spacy-transformers
   python -m spacy download en_core_web_trf

4. HOW TO RUN:
- Place 'create.py' and this 'README.txt' in the same folder.
- Run: python create.py
- The script will output a statistical table in the terminal and 
  save a visualization chart as 'Figure_1.png'.
